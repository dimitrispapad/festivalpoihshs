Νέες Φωνές

Δύο μέρες που η ποίηση δεν “ανεβαίνει” στη σκηνή—κατεβαίνει στην πόλη. Στα σκαλιά, στις αίθουσες, στα βλέμματα. Σε μία ανάσα.

Η ποίηση τρύπωσε στο ωρολόγιο πρόγραμμα, χόρεψε στα περιθώρια των σελίδων, παραβίασε τα διάκενα των τετραδίων.

Κάτι έχει να σου πει. Έλα να την αφουγκραστούμε μαζί.

Μπαίνω στο κάλεσμα Έρχομαι με σχολείο
  • Ρυθμός: εργαστήρια • αναγνώσεις • περιήγηση • συνάντηση
  • Χρόνος: Παρασκευή 5/6 & Σάββατο 6/6
  • Σπίτι: Ρέθυμνο (και ό,τι χωράει μέσα του)

Δεν έρχεσαι για να “είσαι έτοιμος/η”. Έρχεσαι για να δοκιμάσεις.

Τι είναι οι «Νέες Φωνές»

Ένα φεστιβάλ που αγαπά τις πρώτες απόπειρες, τις “μισές” γραμμές, τις σιωπές που γίνονται λόγος. Ένας χώρος όπου η φωνή βρίσκει τόπο.

Σε ποιους ανοίγει

Αν έχεις φωνή — ή αν ψάχνεις τη φωνή σου — είσαι μέσα.

Μαθητές/Μαθήτριες

Χώρος για παιχνίδι με τη γλώσσα, ρυθμό, εικόνες. Χωρίς φόβο.

Εκπαιδευτικοί

Μια ευκαιρία να γίνει η ποίηση πράξη, μέσα και έξω από την τάξη.

Νέοι δημιουργοί

Open call, μικρόφωνο ανοιχτό, εργαστήρια: δοκιμή, όχι τελειότητα.

Κοινό

Έλα να ακούσεις. Να περπατήσεις. Να μείνεις λίγο στην ίδια λέξη.

Δεν χρειάζεται να…

  • έχεις “σωστό” τρόπο να γράφεις
  • έχεις δημοσιεύσει ή εμφανιστεί κάπου
  • νιώθεις έτοιμος/η — αρκεί να είσαι παρών/παρούσα

Το Φεστιβάλ

Όχι “εκδήλωση”. Μια συνάντηση που αφήνει ίχνη.

Η ποίηση δεν είναι μακριά. Είναι στο σώμα της φωνής, στην παύση πριν από τη λέξη, στο “ξαναπες το”.

Οι «Νέες Φωνές» είναι ένα διήμερο όπου η ποίηση γίνεται πράξη: γράφουμε, ακούμε, μοιραζόμαστε, περπατάμε την πόλη σαν να διαβάζουμε έναν ζωντανό στίχο.

Αρχές / Αξίες

  • ασφαλές πλαίσιο (χωρίς ειρωνεία, χωρίς αποκλεισμούς)
  • συμπερίληψη (οι φωνές δεν μοιάζουν—κι αυτό είναι καλό)
  • πειραματισμός (δοκιμές, ρωγμές, ξαναγράψιμο)
  • χαρά (η ποίηση ως παιχνίδι, ως αναπνοή)

Τι μπορεί να σου μείνει

  • μια άσκηση που θα συνεχίσεις σπίτι
  • μια φράση που θα σε ακολουθήσει στον δρόμο
  • μια γνωριμία που θα γίνει συνεργασία
